# Look命令使用指南

## 简介

Look命令是CampusWorld场景中的核心命令之一，允许玩家查看当前环境或特定物品的详细信息。该命令参考了Evennia的设计模式，提供了直观和强大的查看功能。

## 基本语法

```
look [物品名]
```

## 命令别名

- `l` - 简短别名，最常用
- `lookat` - 完整别名
- `examine` - 检查别名

## 使用方法

### 1. 查看当前环境

当不带任何参数时，look命令会显示当前房间的详细信息。

**命令示例：**
```
> look
> l
> lookat
> examine
```

**显示内容：**
- 房间名称和详细描述
- 可用的出口方向
- 房间内的物品列表
- 房间状态信息（类型、容量、光线、温度等）
- 其他在线玩家

**示例输出：**
```
校园广场
====

校园的中心区域，有喷泉和绿树，是学生们聚集的地方。

出口: library, canteen, dormitory
物品: fountain, tree, bench

房间类型: outdoor, 容量: 3/100, 光线: bright, 温度: 25°C

其他玩家: 张三, 李四
```

### 2. 查看特定物品

当指定物品名称时，look命令会搜索并显示该物品的详细信息。

**命令示例：**
```
> look fountain
> l books
> lookat tree
> examine desk
```

**显示内容：**
- 物品名称和详细描述
- 物品类型和当前位置
- 物品状态信息
- 物品特殊属性

**示例输出：**
```
喷泉
==

一个美丽的圆形喷泉，水花四溅，发出悦耳的声音。

类型: decoration
位置: campus
状态: active
```

### 3. 模糊搜索

look命令支持模糊搜索，可以输入物品名称的一部分来查找。

**命令示例：**
```
> look book    # 查找包含"book"的物品
> l foun       # 查找包含"foun"的物品
> lookat tre   # 查找包含"tre"的物品
```

**多匹配处理：**
当找到多个匹配的物品时，系统会显示选择列表：

```
找到多个匹配 'book' 的物品:

1. 书籍 (study_material)
2. 笔记本 (study_material)
3. 教科书 (study_material)

请使用更具体的名称，或使用 'look <编号>' 查看特定物品。
```

## 高级功能

### 1. 智能搜索

look命令使用智能搜索算法，支持：
- 部分名称匹配
- 大小写不敏感
- 自动去重
- 优先级排序

### 2. 上下文感知

命令会根据当前环境调整显示内容：
- 根据房间类型显示不同信息
- 根据时间显示不同状态
- 根据天气显示环境描述

### 3. 权限控制

不同权限的用户会看到不同级别的信息：
- 普通用户：基本描述
- 高级用户：详细状态
- 管理员：完整信息

## 常见问题

### Q: 为什么找不到某个物品？

A: 可能的原因：
1. 物品名称拼写错误
2. 物品不在当前房间
3. 物品名称不完整
4. 物品已被移除

**解决方法：**
- 检查拼写
- 使用模糊搜索
- 查看当前房间的物品列表
- 联系管理员

### Q: 为什么显示的信息不完整？

A: 可能的原因：
1. 权限不足
2. 物品状态异常
3. 系统错误

**解决方法：**
- 检查用户权限
- 重新登录
- 联系技术支持

### Q: 如何处理多个匹配结果？

A: 当出现多个匹配时：
1. 使用更具体的名称
2. 查看物品列表选择
3. 使用编号选择（如果支持）

## 错误信息

### 常见错误信息

1. **"找不到 'xxx'"**
   - 原因：物品不存在或名称错误
   - 解决：检查名称或使用模糊搜索

2. **"场景未运行，无法执行look命令"**
   - 原因：场景未启动
   - 解决：启动场景后重试

3. **"查看失败: xxx"**
   - 原因：系统错误
   - 解决：联系技术支持
