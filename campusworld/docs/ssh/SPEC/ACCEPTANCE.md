# 验收检查表

## 连接验收

- [ ] `ssh localhost -p 2222` 建立 SSH 连接
- [ ] 连接建立后显示欢迎 Banner
- [ ] 服务器监听 0.0.0.0:2222（可配置）
- [ ] 支持 RSA 4096 位主机密钥

## 认证验收

- [ ] 正确用户名密码登录成功
- [ ] 错误密码返回 "Authentication failed"
- [ ] 连续 5 次失败后，锁定 IP 300 秒
- [ ] 锁定期间连接被拒绝

## 会话验收

- [ ] 登录成功后 spawn 到 SingularityRoom
- [ ] 会话超时（auth_timeout）后自动断开
- [ ] 正确执行 `quit` 断开连接
- [ ] 连接断开后释放所有资源

## 命令交互验收

- [ ] `look` 返回当前位置完整信息
- [ ] `who` 返回当前在线用户列表
- [ ] `help` 返回可用命令列表
- [ ] `time` 返回当前时间

## 安全验收

- [ ] 速率限制生效（10 连接/分钟）
- [ ] 登录失败追踪记录准确
- [ ] IP 白名单功能正常
- [ ] 会话数据不泄露到其他会话

## 性能验收

- [ ] 单会话内存占用 < 10MB
- [ ] 50 并发会话正常运行
- [ ] 命令执行响应 < 100ms（不含网络延迟）
